About System Requirements Lab (Can You Run It?)

System Requirements Lab, often referred to as "Can You Run It?" (CYRI), is a website that offers a service for checking whether a user's computer meets the minimum or recommended system requirements for running various software applications, primarily video games.

The website provides a tool that scans your computer's hardware and compares it to the requirements of a specific game or software title. It then provides you with information on whether your system can run the software, and if not, it may suggest hardware upgrades to meet the necessary specifications.
